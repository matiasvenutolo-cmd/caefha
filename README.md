# CAEHFA — Plataforma de Socios (demo)

Prototipo navegable de la nueva plataforma digital de CAEHFA y propuesta comercial, desarrollado por **Pinaro**.

- `index.html` — mockup navegable del sitio renovado (home pública + área privada de socios, Centro Normativo con Asistente IA, Academia, perfil y pagos).
- `propuesta.html` — propuesta comercial con roadmap Jul–Dic 2026 y abono mensual.
- `Propuesta-CAEHFA-Pinaro.pdf` — la propuesta en PDF.
- `og.png` — imagen de previsualización para compartir por link.

Sitio estático, sin build. Se publica directo en Vercel (raíz = `index.html`).

> Nota: para que la imagen de preview aparezca en WhatsApp, `og:image` debe apuntar a la URL absoluta del deploy (ej. `https://<proyecto>.vercel.app/og.png`).
