# CAEHFA — Web renovada + propuesta (demo)

Mockups y propuesta comercial preparados por **Pinaro** para CAEHFA (Cámara de Equipamiento Hospitalario de Fabricación Argentina).

- `index.html` — **la web de CAEHFA renovada**: su sitio institucional modernizado, respetando su identidad real (logo, rojo institucional, Lato), su estructura (Institucional, Empresas y Catálogo, Novedades, Observatorio, Academia, Gobierno, Contacto) y su contenido real (fundada en 1976, ~400 empresas, objetivos, internacionalización, novedades).
- `plataforma.html` — **propuesta de Pinaro**: demo navegable de la nueva plataforma de socios (área privada, pagos, Asistente Normativo con IA, Academia, perfil). Es el valor agregado, separado de la web.
- `propuesta.html` — propuesta comercial: roadmap Jul–Dic 2026 y abono mensual. Enlaza a las dos demos.
- `Propuesta-CAEHFA-Pinaro.pdf` — la propuesta en PDF.
- `og.png` — imagen de previsualización para compartir por link.

Sitio estático, sin build. Raíz (`/`) = la web renovada. Se publica directo en Vercel.

> Nota: para que la imagen de preview aparezca en WhatsApp, `og:image` debe apuntar a la URL absoluta del deploy (ej. `https://caefha.vercel.app/og.png`).
